# 💬 채팅 가이드

CodeJam의 내장 채팅 기능을 사용하여 코드와 함께 팀원들과 실시간으로 소통할 수 있습니다.

---

## 📍 채팅 위치

채팅 아이콘은 화면 우측 하단에 플로팅 버튼으로 고정되어 있습니다.

- **아이콘**: 💬 (Chat Bubble)
- **읽지 않은 메시지**: 빨간색 뱃지로 개수 표시

---

## 💬 채팅 패널 열기/닫기

### 패널 열기

1. 우측 하단의 💬 아이콘 클릭
2. 채팅 패널이 우측 하단에서 위로 확장
3. 읽지 않은 메시지 수가 초기화됨

### 패널 닫기

- 패널 상단의 닫기 버튼(×) 클릭
- 또는 💬 아이콘 다시 클릭

---

## ✍️ 메시지 전송

### 기본 전송

```
1. 패널 하단의 입력창에 메시지 입력
2. Enter 키를 누르거나 전송 버튼 클릭
3. 모든 참가자에게 실시간으로 전송됨
```

### 줄바꿈

- **Shift + Enter**: 메시지 내에서 줄바꿈
- **Enter**: 메시지 전송

### 메시지 제한

- **최소 길이**: 1자
- **최대 길이**: 2000자
- **최대 저장 개수**: 최근 300개 메시지

:::tip
메시지가 300개를 초과하면 가장 오래된 메시지부터 자동으로 삭제됩니다.
:::

---

## 📨 메시지 유형

### 일반 메시지

참가자가 작성한 메시지로, 다음 정보를 포함합니다:

```
[아바타] 닉네임
        메시지 내용
        시간 (HH:MM)
```

### 시스템 메시지

시스템이 자동으로 생성하는 알림 메시지로, 중앙 정렬되며 회색 텍스트로 표시됩니다.

**시스템 메시지 종류:**

- ✅ "Alice님이 입장하셨습니다"
- 🚪 "Bob님이 퇴장하셨습니다"
- 👑 "Charlie님이 방장이 되었습니다"
- ⚡ "David님이 편집자로 변경되었습니다"
- 👁️ "Eve님이 관전자로 변경되었습니다"
- 📄 "index.js 파일이 생성되었습니다"
- 🗑️ "main.py 파일이 삭제되었습니다"

---

## 🔔 알림

### 읽지 않은 메시지 알림

채팅 패널이 닫혀 있는 상태에서 새 메시지가 도착하면:

- 💬 아이콘 우측 상단에 숫자 뱃지 표시
- 패널을 열면 카운트가 0으로 초기화

### 브라우저 알림

현재 브라우저 푸시 알림은 지원하지 않습니다. (소음 방지)

## 💡 활용 팁

### 1. 코드 리뷰 시

```
"15번째 줄 로직 확인 부탁드립니다."
"여기 변수명을 더 명확하게 바꿔볼까요?"
```

### 2. 페어 프로그래밍 시

```
"제가 먼저 함수 구조를 잡아볼게요"
"이제 여기부터 같이 작성해봅시다"
```

### 3. 알고리즘 스터디 시

```
"이 방법으로 시간 복잡도가 O(n²)인데, 개선할 방법 있을까요?"
"DFS로 풀어보면 어떨까요?"
```

## 🔒 개인정보 보호

- 채팅 내용은 서버에 영구 저장되지 않습니다
- 클라이언트 메모리에만 최대 300개 메시지 유지
- 방이 삭제되거나 24시간 후 만료 시 모든 메시지 자동 삭제

:::warning
민감한 개인정보나 비밀번호는 채팅에 입력하지 마세요.
:::
