# 👋 CodeJam 소개

**"복잡한 설정 없이, 링크 하나로 코딩을 시작하세요."**

CodeJam은 개발자들의 가벼운 아이디어 회의, 기술 면접, 그리고 빠른 페어 프로그래밍을 위해 탄생했습니다. 무거운 IDE를 켜거나 복잡한 환경 설정을 할 필요가 없습니다. 브라우저만 있다면 어디서든 실시간 협업이 가능합니다.

---

## 🚀 왜 CodeJam인가요?

### 1. Zero-Config, Login-Free

CodeJam의 가장 큰 철학은 **'속도'** 와 **'간편함'** 입니다.

- **로그인 없음**: 개인정보 입력이나 회원가입 절차 없이, 접속 즉시 고유한 작업 공간(Room)이 생성됩니다.
- **설정 없음**: 언어 설정, 확장을 설치할 필요 없이 바로 타이핑을 시작할 수 있습니다.

### 2. 강력한 동시성 제어 (CRDT)

여러 명이 동시에 코드를 수정해도 충돌이 발생하지 않습니다.

- **Yjs 엔진 탑재**: 분산 시스템 환경에서도 데이터의 일관성을 보장하는 CRDT(Conflict-free Replicated Data Types) 기술을 적용했습니다.
- **실시간 반영**: 팀원이 작성하는 코드가 밀리초(ms) 단위로 내 화면에 동기화됩니다.

### 3. 보안을 위한 자동 파기

- **24시간 후 만료**: 생성된 모든 세션과 코드는 생성 시점으로부터 24시간 뒤에 서버에서 영구적으로 삭제됩니다. 잠깐 쓰고 버리는 메모장처럼 가볍고 안전하게 사용하세요.

## 💻 주요 기능 살펴보기

### 👨‍💻 실시간 협업 (Live Collaboration)

단순히 텍스트만 공유하는 것이 아닙니다. 협업의 '맥락'을 공유합니다.

- **멀티 커서**: 누가 어느 라인을 수정하고 있는지 색상 띠와 이름표로 구분해 줍니다.
- **Follow Mode**: 팀원의 아바타를 클릭하면, 해당 팀원의 시점으로 화면이 고정됩니다. 누군가 코드를 설명할 때 유용하게 활용해 보세요.

### 🛡️ 역할 기반 권한 관리 (RBAC)

방을 생성한 **Host**는 참여자들의 권한을 제어할 수 있습니다.

- **Host (방장)**: 방 설정 변경, 참여자 강제 퇴장, 권한 부여, 코드 읽기/쓰기 가능
- **Editor (편집자)**: 코드 읽기/쓰기 가능 (최대 6명 권장)
- **Viewer (관전자)**: 코드 읽기만 가능 (실수로 코드를 지울 걱정이 없습니다)

## 🛠️ 기술 스택 (Tech Stack)

CodeJam은 최신 웹 기술을 사용하여 빠르고 안정적인 경험을 제공합니다.

| 분류         | 기술                | 사용 목적                         |
| :----------- | :------------------ | :-------------------------------- |
| **Frontend** | React, TailwindCSS  | 직관적이고 반응성 높은 UI 구성    |
| **Editor**   | CodeMirror 6        | 가볍고 확장성 높은 웹 에디터 엔진 |
| **Sync**     | Yjs, Socket.io      | 실시간 데이터 동기화 및 충돌 방지 |
| **Backend**  | NestJS, Redis       | 세션 관리 및 고속 데이터 캐싱     |
| **Infra**    | Naver Cloud, Docker | 안정적인 서비스 배포 및 운영      |
