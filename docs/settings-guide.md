# ⚙️ 설정 가이드

CodeJam의 설정 기능을 활용하여 에디터를 개인 취향에 맞게 커스터마이징할 수 있습니다.

---

## 📍 설정 위치

좌측 사이드바의 설정 탭(⚙️ Settings)에서 모든 환경 설정을 관리할 수 있습니다.

---

## 🔤 폰트 설정

### 폰트 크기 (Font Size)

에디터의 글자 크기를 조절할 수 있습니다.

**설정 방법:**

1. 슬라이더를 좌우로 드래그
2. 또는 입력창에 직접 숫자 입력 (10-30)
3. 에디터에 즉시 반영

**범위:**

- 최소: 10px
- 최대: 30px
- 기본값: 14px

:::tip

- **작은 폰트 (10-12px)**: 한 화면에 많은 코드를 보고 싶을 때
- **중간 폰트 (14-16px)**: 일반적인 코딩 작업에 적합
- **큰 폰트 (18-30px)**: 발표나 화면 공유 시 가독성 향상
  :::

## 👁️ 커서 & 외관 설정

### 원격 커서 표시 (Show Remote Cursors)

다른 참가자의 커서 위치를 실시간으로 표시합니다.

- **ON**: 다른 참가자의 커서가 에디터에 표시됨 (기본값)
- **OFF**: 커서 숨김 (성능 향상)

**언제 끄나요?**

- 참가자가 많아 화면이 복잡할 때
- 혼자 집중해서 작업하고 싶을 때
- 저사양 기기에서 성능을 높이고 싶을 때

### 프로필 표시 (Show Gutter Avatars)

에디터 좌측 여백(Gutter)에 참가자 아바타를 표시합니다.

- **ON**: 각 참가자가 편집 중인 줄에 아바타 표시 (기본값)
- **OFF**: 아바타 숨김

**활용 방법:**

- 아바타를 클릭하면 Follow Mode 활성화
- 해당 참가자가 편집 중인 위치로 자동 이동
- 참가자의 커서를 실시간으로 추적

### 이름표 항상 보기 (Always Show Labels)

커서 위에 참가자 이름을 항상 표시할지 선택합니다.

- **ON**: 커서 위 참가자 이름을 항상 표시
- **OFF**: 마우스 오버 시만 표시 (기본값)

## ⚡ 코드 실행 설정

### 실시간 출력 (Real-time Output)

코드 실행 중 출력 방식을 선택합니다.

- **ON**: 코드 실행 중 스트리밍 방식으로 출력 (기본값)
  - `print()`, `console.log()` 등이 즉시 출력
  - 진행 상황을 실시간으로 확인 가능
- **OFF**: 실행 완료 후 한 번에 출력
  - 최종 결과만 확인하고 싶을 때 유용

**예시:**

```python
# 실시간 출력 ON
for i in range(5):
    print(f"Processing {i}...")
# → 각 줄이 순차적으로 출력됨

# 실시간 출력 OFF
# → 모든 실행이 끝난 후 한 번에 출력
```

## 🎨 테마 설정

### 다크 모드 / 라이트 모드

헤더 우측 상단의 테마 토글 버튼으로 전환할 수 있습니다.

- **아이콘**: 🌙 (다크 모드) / ☀️ (라이트 모드)
- **저장**: localStorage에 자동 저장
- **자동 적용**: 다음 접속 시 이전 테마 자동 적용

**장점:**

- **다크 모드**: 장시간 코딩 시 눈의 피로 감소
- **라이트 모드**: 밝은 환경에서 가독성 향상

---

## 🔄 설정 초기화

### 모든 설정을 기본값으로 복원

설정 탭 하단의 "설정 초기화" 버튼을 클릭하면 모든 설정이 기본값으로 복원됩니다.

**초기화 항목:**

- 폰트 크기: 14px
- 원격 커서 표시: ON
- 프로필 표시: ON
- 이름표 항상 보기: OFF
- 실시간 출력: ON

:::info
초기화는 즉시 적용되며, 확인 대화상자가 표시되지 않습니다.
:::

## 📊 권장 설정

### 상황별 추천 설정

#### 1. 페어 프로그래밍

```
✅ 원격 커서 표시: ON
✅ 프로필 표시: ON
✅ 이름표 항상 보기: ON
✅ 실시간 출력: ON
```

**이유:** 팀원의 위치와 활동을 실시간으로 확인하며 협업

#### 2. 코드 리뷰

```
✅ 원격 커서 표시: ON
✅ 프로필 표시: ON
❌ 이름표 항상 보기: OFF
✅ 실시간 출력: OFF
```

**이유:** 필요할 때만 정보 표시, 코드 가독성 우선

#### 3. 발표 / 화면 공유

```
❌ 원격 커서 표시: OFF
❌ 프로필 표시: OFF
❌ 이름표 항상 보기: OFF
✅ 실시간 출력: ON
📏 폰트 크기: 18-24px
```

**이유:** 깔끔한 화면으로 발표에 집중, 큰 폰트로 가독성 향상

#### 4. 알고리즘 스터디 (다수 참가)

```
❌ 원격 커서 표시: OFF
❌ 프로필 표시: OFF
❌ 이름표 항상 보기: OFF
✅ 실시간 출력: ON
```

**이유:** 참가자가 많을 때 화면 복잡도 감소

## 💾 설정 저장

모든 설정은 브라우저의 **localStorage**에 자동으로 저장됩니다.

**특징:**

- 브라우저를 닫아도 유지
- 같은 브라우저에서 다시 접속 시 자동 적용
- 다른 브라우저나 기기에서는 초기값 사용

---

## 🔒 개인정보 보호

설정 정보는 오직 **로컬 브라우저**에만 저장되며, 서버로 전송되지 않습니다.

**저장 위치:**

```javascript
localStorage.setItem(
  'codejam-settings',
  JSON.stringify({
    fontSize: 14,
    showRemoteCursors: true,
    showGutterAvatars: true,
    alwaysShowLabels: false,
    realtimeOutput: true,
    theme: 'dark',
  }),
);
```

---

## 🆘 문제 해결

### 설정이 저장되지 않음

**해결 방법:**

1. 브라우저의 localStorage 지원 확인
2. 시크릿 모드나 프라이빗 브라우징 사용 중인지 확인
3. 브라우저 쿠키 및 사이트 데이터 설정 확인

### 설정이 적용되지 않음

**해결 방법:**

1. 브라우저 새로고침 (F5)
2. 설정 초기화 후 다시 설정
3. 브라우저 캐시 삭제
